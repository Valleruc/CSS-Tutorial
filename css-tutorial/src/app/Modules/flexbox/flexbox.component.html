Introcution to Flexbox

Flexbox is a layout model that allows you to distribute items in a 2-Dimensional space along 1 axis. Using either the Main Axis or the Cross Axis, Items can be aligned along them with various amounts of spacing and positioning. Bellow there are examples of various attributes of Flexbox and their many layouts.
